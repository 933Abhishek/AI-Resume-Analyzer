# 🚀 AI Resume Analyzer

A web-based application that analyzes resumes and provides ATS-based feedback.

---

## 🔥 Features
- Upload Resume (PDF)
- Extract Resume Text
- Detect Skills
- Generate Score
- Provide Suggestions

---

## 🛠 Tech Stack
- Node.js
- Express.js
- HTML, CSS, JavaScript

---

## 📂 Project Structure
AI-Resume-Analyzer/
├── index.html
├── server.js
├── package.json
├── README.md

---

## ⚙️ Run Locally

npm install  
node server.js  

Open: http://localhost:5000

---

## 🎯 Output
- Resume Score
- Skills Detected
- Suggestions
