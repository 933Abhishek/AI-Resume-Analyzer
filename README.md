# 🚀 AI Resume Analyzer

A web-based application that analyzes resumes and provides intelligent feedback based on ATS (Applicant Tracking System) logic.

---

## 🔥 Features

* 📄 Upload Resume (PDF)
* 🧠 Extract Text from Resume
* 🛠 Detect Technical Skills
* 📊 Generate Resume Score (ATS-based)
* 💡 Provide Smart Improvement Suggestions

---

## 🧠 How It Works

* Extracts text from PDF using `pdf-parse`
* Uses keyword matching to detect skills
* Applies rule-based logic to evaluate resume quality
* Generates suggestions based on missing sections and content

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/your-username/ai-resume-analyzer.git
```

2. Navigate to project folder

```
cd ai-resume-analyzer
```

3. Install dependencies

```
npm install
```

4. Run the server

```
node server.js
```

5. Open in browser

```
http://localhost:5000
```
